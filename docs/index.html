nzilbb.labbcat.server Documentation

nzilbb.labbcat.server.jar

This is java library for implementing the parts of the LaBB-CAT web application which implement the nzilbb.ag.IGraphStoreQuery and related interfaces, and interact with the nzilbb.ag object model generally.

Classes include selected servlets, and data-store access classes.

Detailed JavaDoc object model documentation is available here.

nzilbb.labbcat.server Servlet API

This library includes various servlets that implement a JSON-based web API.

For API function documentation, see the nzilbb.labbcat.server.servlet summary.
