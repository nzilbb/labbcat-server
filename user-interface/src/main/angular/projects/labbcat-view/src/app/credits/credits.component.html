LaBB-CAT (formerly known as ONZE Miner) was designed and written by Robert Fromont and Jen Hay for the New Zealand Institute of Language, Brain and Behaviour (NZILBB) at the University of Canterbury, Christchurch, New Zealand.

Many thanks to Dan Villarreal of the University of Pittsburgh for mounting contributions to LaBB-CAT's user experience.

The NLP parser was provided by The Stanford Natural Language Processing Group.

The CELEX database is available from the Linguistic Data Consortium.

LaBB-CAT can be configured to use the Hidden Markov Model Toolkit (HTK) provided by the Cambridge University Engineering Department. Optionally, the pre-trained Penn Phonetics Lab Force Aligner models (P2FA - Jiahong Yuan and Mark Liberman. 2008. Speaker identification on the SCOTUS corpus. Proceedings of Acoustics '08.) may be used for alignment in conjunction with HTK.

Most icon images are from Open Iconic, the Tango Desktop Project, and others are from Oliver Scholtz's  Human-O2 Icons.

The font is the Atkinson Hyperlegible Font published by the Braille Institute.

In-situ editing of online help and other HTML made possible by CKEditor.

Conversion to PDF made possible by iText ®.

Browser-based audio recording achieved using components by Matt Diamond and Chris Wilson.

Browser-based script editing achieved using CodeMirror by Marijn Haverbeke and others.

Syntactic parse tree rendering achieved using vis.js by almende.

Acoustic analysis is done using Praat by Paul Boersma and David Weenink.

Emu in-situ annotation visualisation by EMU-webApp by Raphael Winkelmann and Georg Räß.

Query language parsing uses ANTLR by Terence Parr and many others.

Process with Praat can be configured to use FastTrack by Santiago Barreda; to cite FastTrack please use: Barreda, S. (2021). Fast Track: fast (nearly) automatic formant-tracking using Praat. Linguistics Vanguard, 7(1). https://doi.org/10.1515/lingvan-2020-0051

LaBB-CAT relies heavily on many Apache components, including Tomcat, Ant, Maven, and Commons.

LaBB-CAT's user interface is increasingly implemented using the Angular framework.
